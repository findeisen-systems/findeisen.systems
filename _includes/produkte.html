Managed VPN Server

Surfen Sie in Zukunft mit ihrem Android Telefon oder ihrem Notebook über einen sicheren Kanal wenn Sie sich in fremden Netzen befinden. Mit einem OpenVPN Server von Fonion.net ist dies sehr einfach möglich. Wir verwalten den Server für Sie. 40TB Datentransfer ist enthalten.

Managed XMPP Server

Gerne stellen wir Ihnen einen eigenen XMPP Server zur Verfügung. Sie bekommen hierfür auf Wunsch einen DNS Namen wie FIRMA.xmpp.network oder Sie richten einen Namen in Ihrem DNS ein. SSL Verschlüsselung ist selbstverständlich inklusive.
