Über Fonion

Die Geschichte über com.puting.de zu FONION.

  • 2002

    com.puting.de wird gegründet

    Johannes Findeisen gründet im jahr 2002 das Unternehmen com.puting.de. Am Anfang auf Web-Entwicklung spezialisiert wird aus dem Unternehmen schnell ein Beratungs-Unternehmen für Linux im Enterprise Umfeld.

  • 2002 - 2017

    com.puting.de lebt!

    Durch die hohe Nachfrage an Linux Unterstützung in Unternehmen spezialisiert sich com.puting.de zunehmend auf Linux im Server Bereich. Auch Software-Entwicklung mit C, PHP, JAVA und Python wird für kleine bis große Unternehmen realisiert. Open Source Software steht dabei immer im Mittelpunkt.

  • 2018

    Reboot! Fonion wird geboren.

    Viele Bereiche der Informationsverarbeitung haben sich stark verändert und es hat viel Innovation stattgefunden. Anforderungen sind Heute andere. Mit Fonion orientiert Johannes sich neu und der Fokus liegt auf Lösungen für sichere mobile Kommunikation. Server Lösungen spielen hierbei noch immer eine große Rolle allerdings auch mobile Geräte wie Smartphones.
